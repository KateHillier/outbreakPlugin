Outbreak

For when your infectious love of overlays just cannot be... contained

@HackerYou

How to:

1. Link the 'outbreak.js' file to your HTML file.

2. Set up your Overlay. I recommend a div with an id# on which you can call the main .js file. Within the div link your image and create a second div for the Overlay:

<div id="baseImg">
	<img src="">
	<div class="overlay"></div>
</div>

3. Setup your div to contain the Outbreak. Be sure to set the width and height of your img (or the element that you are applying the overlay to) in your CSS file:

#baseImg {
	position:relative; 
	top:0px;
	margin: 0 auto;
	width: 500px;
	height: 350px;
	overflow: hidden;
}

4. Set the styles for your Outbreak infection - copy & paste the following to your CSS:

.spot {
	position: absolute;
	border-radius: 50%;
	color: #D12026;
	opacity: 0;
	/*-webkit-animation-name: infection;*/
	-webkit-animation-duration: 3s;
	-webkit-animation-fill-mode: forwards;
	-webkit-animation-timing-function: ease-in;
}	

@-webkit-keyframes infection {
	0% {
		box-shadow: 0 0 0 0 #D12026;
	}

	100% {
		box-shadow: 0 0 0 75px #D12026;
	}
}

5. In your 'main.js' file set the div ( or element) on which you will call the Overlay (**best not to use the img itself) & set the outbreak number to the number of spots you require to fill your Overlay:

$(function() {
	$('#baseImg').outbreak(100);
});
